# 🌈 Mood Assistant AI

An interactive AI chatbot that understands user moods and responds accordingly.

## 🚀 Features
- 💬 AI-powered chatbot (Chatling integration)
- 😊 Mood-based interaction (happy, sad, stressed)
- 🎨 Modern responsive UI
- ⚡ Fast and simple user experience

## 🛠️ Technologies Used
- HTML
- CSS
- Chatling AI

## 📌 How It Works
1. User selects or types their mood
2. Chatbot processes input
3. AI responds with appropriate message
